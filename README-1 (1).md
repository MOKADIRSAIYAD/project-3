# Collection Manipulator (Simple Version)

## 📌 Project Description

This is a simple Python-based menu-driven program that allows users to
perform operations on different data structures: - List - Tuple - Set -
Dictionary

It is designed for beginners to understand how Python collections work
interactively.

------------------------------------------------------------------------

## 🚀 Features

### 1. List Operations

-   Add items
-   Remove items
-   Display list

### 2. Tuple Operations

-   Display tuple
-   Find length
-   Find max and min values

### 3. Set Operations

-   Union
-   Intersection
-   Difference

### 4. Dictionary Operations

-   Add key-value pairs
-   Delete key
-   Display dictionary

------------------------------------------------------------------------

## 🛠️ Technologies Used

-   Python (Basic Concepts)
-   Loops & Conditional Statements
-   Data Structures

------------------------------------------------------------------------

## ▶️ How to Run

1.  Install Python on your system
2.  Copy the code into a `.py` file (e.g., `collection.py`)
3.  Run the program:

``` bash
python collection.py
```
